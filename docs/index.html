Radon

Radon is a dynamically typed language which compiles to bytecode and is executed on a virtual machine.

It's in very early development, and the source is hosted on GitHub. If anyone wants to make a pull request, that's great :) There's a huge TODO list in the README.

Here are some examples of its various features:

Basic syntax

                            
v = fib(20)
print(v)

a = (2 ^ v) - 1

# hoisted
fib(n) = match n where
    | 0 -> 1,
    | 1 -> 1,
    | _ -> fib(n - 1) + fib(n - 2)

Models

                            
vector = model(x, y)

vector.add(other) = {
    self.x = self.x + other.x
    self.y = self.y + other.y
}

a = vector(5, 2)
b = vector(-1, 8)
c = a.add(b)

Control flow

                            
x = 5
y = 10

if x > y then print("hello")

if true then {
    print("x")
} else if false then {
    print("y")
} else {
    print("z")
}

Iteration

                            
x = 0

while (x < 10) {
    print(x)
    x = x + 1
}

for (i = 0; i < 10; i = i+1) print(i)

loop {
    print("help, i'm stuck in an infinite loop")
}
